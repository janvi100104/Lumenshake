# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, inclusive,
and harassment-free experience for everyone.

## Expected Behavior

- Be kind, constructive, and professional.
- Respect different viewpoints and lived experiences.
- Assume good intent and ask clarifying questions before escalating.
- Focus feedback on code, behavior, and outcomes, not people.

## Unacceptable Behavior

- Harassment, hate speech, discrimination, or intimidation.
- Personal attacks, insults, or hostile communication.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.

## Scope

This Code of Conduct applies in all project spaces, including:
- GitHub issues and pull requests
- Discussions, docs, and reviews
- Public spaces when representing LumenShake

## Reporting

If you experience or witness unacceptable behavior, report it privately:

- Email: `janvisinghal10@gmail.com`
- Subject: `LumenShake Code of Conduct Report`

Please include as much context as possible. Reports are handled confidentially.

## Enforcement

Maintainers may take any action they consider appropriate, including:
- Warning
- Comment moderation/removal
- Temporary ban
- Permanent ban

Enforcement decisions are made to protect the health of the community.
